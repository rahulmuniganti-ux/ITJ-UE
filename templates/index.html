Welcome to ITJ-UE Portal

ITJ-UE Portal

The ITJ-UE (Intelligent Timetable Generator – University Exams) automates extraction and management of examination timetable data from official PDF documents, reducing manual work and improving accuracy.

About JNTUH

Jawaharlal Nehru Technological University Hyderabad (JNTUH) is a premier technical university in India, established in 1972. It conducts examinations for numerous affiliated colleges and publishes detailed exam timetables in PDF format.

About ITJ-UE Project

This project extracts exam details such as dates, sessions, branches, and subjects from unstructured PDFs and converts them into structured data for efficient management.

Key Features

✔ Upload multiple timetable PDFs
✔ Automatic data extraction
✔ Consolidated timetable generation
✔ View timetable in web interface
✔ Download Excel reports
✔ Manage upload history

Project Objective

The main objective is to automate timetable processing and provide a user-friendly system that converts complex PDF data into structured formats, reducing errors and saving time.
